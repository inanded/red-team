# Code of conduct

This project adopts the Contributor Covenant, version 2.1, as its code of conduct.

The full text is published at https://www.contributor-covenant.org/version/2/1/code_of_conduct/ and is incorporated here by reference.

## Scope

The code of conduct applies in all project spaces, and also when an individual represents the project or its community in public spaces.
