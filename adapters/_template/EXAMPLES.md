# Examples

Short, redacted examples of findings this adapter has produced against real codebases. One example per override is a good target.

Each example is structured like a finding:

```
### <short title>
- Persona: <persona-name>
- File (redacted): <path>
- Quoted code (redacted): ...
- Verdict: EXPLOITABLE
- Fix: <one line>
```

Keep the list focused. The goal is to give contributors a sense of what the adapter catches, not a complete backlog.
